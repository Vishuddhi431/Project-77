This is my first project in Visual Studio Code. Open the console to see my work.

Visual Studio Code is an IDE (Integrated Developement Environment) used by coders to write and test code at the same time

The console screen displays the output of the code. To access the console screen, open any website, right click, and choose inspect. Then navigate to the console tab. We use the command console.log() to print text onto the console screen.
